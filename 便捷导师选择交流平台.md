# 便捷导师选择&交流平台

## background

有很多刚上大二的学弟学妹会对导师选择这个重要的决定感到十分迷茫。对学校老师的不熟悉、对自己感兴趣的研究方向的迷茫、以及对与长辈社交的恐惧等，都会让这个过程变得十分煎熬。会有一些学弟学妹来找我咨询导师选择的相关事宜，但我发现我只能推荐少数几个“人尽皆知”的导师以及我自己的导师，并不能给他们有效的帮助，因此我想到能否实现这样一个平台，在方便同学寻找合适导师的同时，也帮一些相对“冷门”的导师获得较好的生源。

## function

### 1. 面向不同群体的用户界面

初步计划是完成面向学生和面向导师的两套用户界面，拥有不同的权限。未来可能会加入班主任以及辅导员的角色，以方便他们动态掌握了解同学的学术情况。

### 2. 咨询查询

“导师用户”可以完善自己的个人信息、提出对申请学生的要求（如GPA、必须学习的课程等）、介绍当前的科研情况以及未来的计划等。这些内容会合理存储起来，方便“同学用户”进行查询。“同学用户”可以直观的了解到学校内不同的导师，方便自己的选择。

### 3. 讨论版

在讨论版中，大家可以对自己遇到的问题进行讨论，比如说“我对xxx感兴趣，但是我没有找到相应的研究方向，请问哪一个比较接近”等等。已经进组的同学也可以在这里留下自己的意见，为学弟学妹指引正确的方向。

### 4. 单独聊天

同学与同学、同学与老师之间也可以直接通过本平台进行交流，相较于邮件更加便捷，相对于微信不那么private。当然它也可以用来互相留联系方式。
